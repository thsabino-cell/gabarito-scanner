Exportação, integrações e preferências

Chave de API — Anthropic (Claude IA)
Necessária para escanear cartões com IA
Obtenha sua chave em console.anthropic.com → API Keys. Ela é salva localmente no dispositivo e nunca é enviada para servidores externos.
⚠️ Sem chave configurada, o escaneamento roda em modo demonstração com respostas simuladas (aleatórias). Configure a chave para leitura real com IA.

Google Sheets — Sincronização em Tempo Real
Sincronização via Webhook
Os dados são enviados diretamente para sua planilha Google Sheets usando Apps Script.
Obtida em: Google Sheets → Extensões → Apps Script → Implantar → Novo Implantação
📋 Como configurar (passo a passo)
1. Importe o arquivo gabarito_scanner_sheets.xlsx no Google Sheets
2. Vá em Extensões → Apps Script
3. Apague o código existente e cole o script da aba ⚙ Script da planilha
4. Execute a função instalarWebhook() e copie a URL gerada
5. Cole a URL no campo acima e clique em Sincronizar tudo

Instruções de Uso
Como escanear um cartão
1. Cadastre os alunos na página Alunos
2. Crie uma avaliação em Avaliações
3. Configure as respostas corretas em Gabaritos
4. Vá em Escanear, selecione o aluno e a avaliação
5. Tire uma foto do cartão-resposta preenchido
6. Clique em Analisar com IA
Sistemas de pontuação
Percentual: converte acertos em nota 0–10
Por pontos: 1 ponto por questão correta
Ponderado: usa a nota máxima da avaliação
Binário: ≥60% = Aprovado, <60% = Reprovado
Instalar como Aplicativo
Gabarito Scanner
Instale no celular para acesso rápido, sem precisar do Chrome

1. Clique em Instalar no dispositivo acima
2. Ou toque nos 3 pontos ⋮ do Chrome
3. Selecione "Adicionar à tela inicial"
4. Confirme a instalação
iPhone (Safari)
1. Abra no Safari (não Chrome)
2. Toque em Compartilhar (ícone ↑)
3. Selecione "Adicionar à Tela de Início"
4. Confirme com Adicionar
