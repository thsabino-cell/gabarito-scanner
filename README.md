# Gabarito Scanner — Deploy GitHub Pages

## Como publicar em 3 passos (HTTPS gratuito):

### Passo 1 — Criar repositório
1. Acesse https://github.com e faça login (ou crie conta)
2. Clique em "New repository"
3. Nome: `gabarito-scanner`
4. Marque "Public"
5. Clique "Create repository"

### Passo 2 — Fazer upload dos arquivos
1. Na página do repositório, clique "uploading an existing file"
2. Arraste os arquivos `index.html` e `.nojekyll` para a área de upload
3. Clique "Commit changes"

### Passo 3 — Ativar GitHub Pages
1. Vá em Settings → Pages
2. Em "Source", selecione "Deploy from a branch"
3. Branch: "main", pasta: "/ (root)"
4. Clique "Save"
5. Aguarde 2 minutos e acesse: https://SEU-USUARIO.github.io/gabarito-scanner

## Resultado
- URL HTTPS permanente e gratuita
- Câmera ao vivo funcionando
- Instalável como PWA no Android
- API Key salva permanentemente via IndexedDB
